# Electric Vehicle Charge and Range Analysis 🚗⚡

## Project Overview
This project analyzes Electric Vehicle (EV) adoption, charging patterns, and vehicle range using data visualization.  
The goal is to help understand EV growth trends, distribution across regions, and comparison between different EV types.

The project integrates **data analytics, visualization, and web deployment** using Tableau and Flask.

---

## Technologies Used

- Python
- Flask (Web Integration)
- Tableau Public (Data Visualization)
- MySQL (Data Storage)
- HTML / CSS (Frontend UI)

---

## Project Features

- EV distribution analysis across counties
- EV adoption trends over model years
- Comparison of different EV types
- Interactive Tableau dashboard
- Web integration using Flask
- Professional UI for visualization display

## How to Run the Project

1. Clone the repository

git clone https://github.com/yourusername/EV-Charge-Range-Analysis.git

2. Navigate to the project folder


cd EV_Project


3. Install dependencies


pip install flask


4. Run the Flask application


python app.py


5. Open the browser and visit


http://127.0.0.1:5000
